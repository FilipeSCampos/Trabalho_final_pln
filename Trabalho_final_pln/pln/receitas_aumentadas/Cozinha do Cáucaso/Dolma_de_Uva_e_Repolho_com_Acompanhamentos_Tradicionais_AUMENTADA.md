# Dolma de Uva e Repolho com Acompanhamentos Tradicionais

**Fonte Original:** Azerbaijão/Azeri.docx

## Informações Gerais
| Item | Valor |
| :--- | :--- |
| Porções | 4 a 6 pessoas |
| Tempo de Preparo | 90 minutos |
| Tempo de Cozimento | 60 minutos |

## Ingredientes (Augmentados com Quantidades Médias Brasileiras)

| Ingrediente | Quantidade |
| :--- | :--- |
| Folhas de uva | 200g |
| Folhas de repolho | 300g |
| Carne moída (boi ou cordeiro) | 500g |
| Arroz branco | 1 xícara (200g) |
| Cebola picada | 1 unidade média |
| Alho picado | 3 dentes |
| Tomate picado | 2 unidades médias |
| Salsinha fresca picada | 1/2 xícara |
| Hortelã fresca picada | 1/4 xícara |
| Pimenta do reino | a gosto |
| Sal | a gosto |
| Azeite de oliva | 4 colheres de sopa |
| Suco de limão | 2 colheres de sopa |
| Água | suficiente para cobrir na panela |
| Folhas de louro | 2 folhas |
| Canela em pau | 1 pedaço pequeno |
| Pimenta vermelha em pó (opcional) | 1 colher de chá |

## Modo de Preparo Sugerido

Primeiramente, prepare as folhas de uva e repolho: se forem frescas, escalde rapidamente em água fervente para amolecer e facilitar o enrolar. Em uma tigela, misture a carne moída com o arroz lavado e escorrido, cebola, alho, tomate, salsinha, hortelã, sal, pimenta, e azeite. Misture bem para incorporar os temperos. Para montar os charutos, coloque uma pequena porção do recheio em cada folha, dobre as laterais e enrole firmemente para formar os dolmas. Disponha os charutos em uma panela, organizando em camadas. Adicione água até cobrir, o suco de limão, as folhas de louro e a canela em pau para aromatizar. Cozinhe em fogo baixo por aproximadamente 45 minutos até que o arroz e a carne estejam cozidos e o caldo reduzido. Após o cozimento, deixe descansar tampado por 15 minutos antes de servir. Sirva acompanhado de coalhada seca ou iogurte com hortelã para um toque tradicional.
