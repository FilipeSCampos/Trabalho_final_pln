# Menu Completo Brasileiro Gourmet

**Fonte Original:** Romenia/Romenia 10.04.docx

## Informações Gerais
| Item | Valor |
| :--- | :--- |
| Porções | 4 a 6 pessoas |
| Tempo de Preparo | 90 minutos |
| Tempo de Cozimento | 120 minutos |

## Ingredientes (Augmentados com Quantidades Médias Brasileiras)

| Ingrediente | Quantidade |
| :--- | :--- |
| Pirarucu defumado | 400g |
| Banana da terra | 2 unidades médias |
| Tapioca em farinha | 1 xícara (120g) |
| Polpa de açaí congelada | 100g |
| Tomate para semente (para o caviar) | 150g |
| Limão siciliano | 1 unidade (raspas) |
| Alho-poró | 2 talos médios |
| Batata | 2 unidades médias |
| Óleo de dill | 50ml (pode usar óleo vegetal com dill seco) |
| Óleo para fritura (para crispy de alho-poró) | 300ml |
| Picanha | 1,2kg |
| Mandioca (macaxeira) | 1kg |
| Tomate cereja | 200g |
| Mini cenouras | 150g |
| Aspargos | 200g |
| Caldo para molho demi-glace (concentrado) | 300ml |
| Sal | a gosto |
| Pimenta-do-reino moída na hora | a gosto |
| Azeite de oliva extra virgem | 50ml |
| Manteiga | 50g |
| Gelatto de pistache | 500ml ou 4 bolas grandes |
| Açúcar para calda de caramelo | 150g |
| Creme de leite fresco (para calda e crocante) | 100ml |
| Pistache triturado | 70g |
| Sal marinho para caramelo salgado | 1 colher de chá |

## Modo de Preparo Sugerido

Etapa 1: Corte o pirarucu defumado em cubos pequenos. Descasque e corte a banana da terra em cubinhos para fazer o tartare, tempere com sal, pimenta e um pouco de suco de limão siciliano para evitar escurecimento. Prepare os crostini de tapioca: hidrate a farinha de tapioca, forme pequenas panquecas e doure em frigideira até ficarem crocantes. Cubra com uma camada fina de polpa de açaí sem açúcar. Faça o caviar de tomate extraindo as sementes dos tomates e coagule em pequenas pérolas, temperando levemente com sal. Finalize com raspas de limão siciliano por cima para um toque cítrico.

Etapa 2: Lave e corte o alho-poró em rodelas finas, separe algumas partes para o crispy e as outras para a sopa. Descasque e corte as batatas em cubos pequenos. Refogue o alho-poró com manteiga, acrescente as batatas e cubra com caldo ou água, cozinhe até ficarem macias. Bata no liquidificador para obter um creme aveludado. Prepare o óleo aromatizado com dill fresco (ou seco) aquecendo o óleo e deixando-o infundir. Frite as rodelas finas de alho-poró em óleo quente até ficarem crocantes, escorra em papel absorvente.

Etapa 3: Prepare a picanha temperando com sal grosso e pimenta-do-reino, grelhe na churrasqueira ou brasa até o ponto desejado. Descasque e corte a mandioca em lâminas finas para o mil folhas, cozinhe até ficarem macias e monte camadas intercaladas com manteiga, levando ao forno para gratinar. Confit os tomates cereja lentamente em azeite com ervas por cerca de 30 minutos. Cozinhe as mini cenouras e os aspargos no vapor, temperando com sal e manteiga. Prepare o molho demi-glace reduzindo o caldo concentrado até obter a consistência desejada.

Sobremesa: Sirva o gelatto de pistache com a calda de caramelo salgado, preparada caramelizando o açúcar, adicionando creme de leite e sal marinho ao final. Finalize com crocante de pistache triturado por cima para textura.

Essa sequência, devidamente sincronizada, permite um menu equilibrado, com preparos paralelos e apresentação sofisticada para 4 a 6 pessoas.
